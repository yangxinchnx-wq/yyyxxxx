# SoloForge IDE

Front-end IDE for the SoloForge AI society system. Bundles a Vite + React UI with a small Express server providing Git workflow, system telemetry, custom rules management, and webhook channel testing.

## Run Locally

**Prerequisites:**  Node.js (LTS)


1. Install dependencies:
   `npm install`
2. (Optional) Copy `.env.example` to `.env` and set `APP_URL` if you need self-referential links
3. Run the app:
   `npm run dev`
